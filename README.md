SystemNodeJs
Project Overview

SystemNodeJs is a simple Node.js application that allows users to create, update, and delete news posts. It provides a RESTful API for managing users and their news posts. The project is built using Express.js for the backend, MongoDB for the database, and Mongoose for object data modeling (ODM).
Features

    User registration and authentication
    CRUD (Create, Read, Update, Delete) operations for news posts
    User roles and permissions management

Technologies Used

    Node.js
    Express.js
    MongoDB
    Mongoose
